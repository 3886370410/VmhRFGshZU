# 项目简介

## 核心功能
- 基于协同过滤算法的首页课程推荐
- 用户注册与登录
- 课程视频与资料浏览
- 免费课程与课程公告
- 排行榜与积分礼品
- 课程作业提交与批改
- 论坛讨论与举报记录
- 系统管理与个人中心操作

## 适用场景
- 在线教育平台
- 学习资源共享社区
- 教育机构内部教学管理

## 技术特点
- 采用B/S架构，支持多用户同时访问
- MVC模式，代码结构清晰，易于维护
- 使用Java、SpringBoot、Vue等技术栈，保证系统稳定性和扩展性
- 支持前后端分离，便于开发和部署
- 使用Maven进行项目管理，提高开发效率
